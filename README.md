# rspack_web
使用rspack搭建一个前端标准应用，集成antd、antv等最新版本，提供可视化方向的前端工程模版

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
